 Douban Notion Book Importer v1.0.0

一个简单干净的小工具，用 ISBN 自动从豆瓣抓取图书信息，并一键写入 Notion Database。

---

## 功能

通过 ISBN 自动查询豆瓣图书信息

自动解析：书名、作者、译者、出版社、出版日期

自动写入到指定的 Notion Database（不会修改额外字段）

内置 GUI，不需要命令行

数据字段完全与你个人 Notion 数据库结构兼容

支持连续导入多本书


## 使用方式

下载本页的 NotionBookImporter.exe

打开工具

输入：

Notion Token

Database ID

ISBN

点击“导入到 Notion”即可
